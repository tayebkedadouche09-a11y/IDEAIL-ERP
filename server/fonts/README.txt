Arabic Font Support for PDF Generation

Required fonts:
- Amiri-Regular.ttf - Download from: https://github.com/aliftype/amiri/raw/main/fonts/ttf/Amiri-Regular.ttf
- Amiri-Bold.ttf - Download from: https://github.com/aliftype/amiri/raw/main/fonts/ttf/Amiri-Bold.ttf

These fonts are needed for proper Arabic text rendering in PDF documents.
If the fonts are not available, the system will fall back to default fonts.
